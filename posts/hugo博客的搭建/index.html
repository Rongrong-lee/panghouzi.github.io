Hugo博客的搭建

January 17, 2020

安装 hugo

mac 打开 item 或者终端，输入命令行

ls ~
brew install hugo

完成后，执行

hugo version

可以查看 hugo 的版本以及是否安装成功

  1. 创建以一个新hugo的网址，必须加一个后缀用来区分后面的另一个仓库 
    hugo new site 仓库名.github.io-creator(一般是固定的)

  1. 添加主题

    cd 仓库名.github.io-creator
git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
echo 'theme = "ananke"' >> config.toml

增加新的内容

hugo new posts/DDDDDD(可以自己命名).md

  1. 执行命令行后自动转入vscode

    code .
  • 注意vscode右下角文本需要调成Markdown模式如图

  1. 如图右侧contents/posts下的文件鼠标右键用终端打开

  2. 将true去掉变成false

  3. 启动hugo服务器

  • 注意：启动服务器是一定要回到根目录下

    hugo server -D
  1. 新建一个终端，hugo不能删除。执行 
    hugo

将博客上传到网上

  1. 建立一个文件将public进行忽略,命名为.gitignore在里面编辑/public/

  2. 进入public

    cd ./public/

  3. 在./public/内运行

    git init
git add .
git commit -v

  4. 去github网站自己的账号下新建一个项目，项目名称有讲究必须是:用户名.github.io,然后执行命令

    git remote add origin git@github.com:用户名.github.io.git
   
git push -u origin master

在github网站上刷新页面，进入刚刚建立的项目中，点击settings，拉到页面尾部找到github pages，选择master。得到网址
