Hugo博客的搭建

安装 hugo mac 打开 item 或者终端，输入命令行 ls ~ brew install hugo 完成后，执行 hugo version 可以查看 hugo 的版本以及是否安装成功 创建以一个新hugo的网址，必须加一个后缀用来区分后面的另一个仓库 hugo new site 仓库名.github.io-creator(一般是固定的) 添加主题 cd 仓库名.github.io-creator git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke echo 'theme = "ananke"' >> config.toml 增加新的内容 hugo new posts/DDDDDD(可以自己命名).md 执行命令行后自动转入vscode code . 注意vscode右下角文本需要调成Markdown模式如图 如图右侧contents/posts下的文件鼠标右键用终端打开 将true去掉变成false 启动hugo服务器
